## 常见问题排查

| 现象 | 可能原因 | 解决 |
|------|----------|------|
| 登录后白屏 | 服务端没启动 | `node server\start.js` |
| 登录报"服务器连接失败" | 同上 | 同上 |
| 文件树点击没反应 | 硬编码文件名与项目目录不匹配，正常 | 不影响核心功能 |
| AI 发送报错 | Claude proxy 未运行 | 不影响其他功能 |

## 快速验证命令

```powershell
# 验证服务端正常
curl http://localhost:3001/api/health
# 期望: {"status":"ok"}

# 验证登录 API
curl -X POST http://localhost:3001/api/auth/login -H "Content-Type: application/json" -d "{\"username\":\"admin\",\"password\":\"admin123\"}"
# 期望: {"accessToken":"...","expiresIn":900,"user":{"id":"admin-001","username":"admin","role":"admin"}}

# 验证未认证被拒
curl -X POST http://localhost:3001/api/auth/login -H "Content-Type: application/json" -d "{\"username\":\"admin\",\"password\":\"wrong\"}"
# 期望: {"error":"invalid_credentials","message":"用户名或密码错误"}
```
